# Privacy Policy Draft for AharSetu Last updated: July 2, 2026 ## 1. Introduction AharSetu is a diet and wellness application that helps users manage health details, prakriti information, daily health check-ins, meal recommendations, notifications, and profile settings. By using AharSetu, you agree to this Privacy Policy. ## 2. Information We Collect We may collect and store the following information: ### Account Information - Name - Email address - Authentication provider details ### Profile and Health Information - Age - Gender - Height - Weight - Date of birth - Time of birth - Prakriti assessment results - Diet preference - Calorie targets - Goal weight ### Daily Usage and Wellness Information - Daily health check answers - Digestive and food preference information - Daily routine logs - Diet recommendations generated by the app ### Files and Media - Profile photo uploaded by the user ### Device / App Preferences - Notification settings - Notification reminder times - Cached local settings needed for app functionality ## 3. How We Use This Information We use collected data to: - create and manage user accounts - personalize diet and wellness recommendations - calculate calorie and goal information - save user progress and profile settings - send meal reminder notifications - improve the app experience ## 4. Data Storage Your data may be stored using: - Firebase Authentication - Cloud Firestore - Firebase Storage - local device storage for non-sensitive settings ## 5. Data Sharing We do not sell personal data. We may rely on service providers such as Firebase and Google Sign-In to operate the app. These providers may process data as required to deliver authentication, cloud storage, and notification services. ## 6. Data Security We take reasonable measures to protect user data, including: - authenticated account access - user-specific cloud data structure - controlled cloud storage access - limited local storage use for non-sensitive preferences No method of electronic storage or transmission is completely secure. ## 7. Data Retention We retain user data while the account remains active or as needed to provide app functionality, comply with legal obligations, resolve disputes, and enforce agreements. ## 8.
